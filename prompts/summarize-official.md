# Official / Policy Summary Prompt

For items in the **Official / Policy** category (Amazon / Walmart official pages,
SP-API release notes, Amazon Ads library). These are policy authority.

Per item, write three item-specific lines (translated per `config.language`):

1. **What happened** — the concrete change or announcement, with its **effective
   date** if the source gives one. Name the specific API / program / policy.
   Lead with the change, not "the page shows…".
2. **Why it matters to sellers** — exactly what is affected: account, SP-API
   integration, ads, logistics/FBA, pricing display, compliance. Be specific to
   *this* change, not a generic line.
3. **One concrete action** — a specific check or change tied to this item.

Rules:
- These override community/media claims. State them as authoritative.
- If the item is flagged `thin` or the body is just a doc index / JS shell:
  do NOT invent entries or write "go to the changelog and look yourself".
  Write one honest line — "source returned only an index/JS shell; no
  substantive update captured, confirm at the link" — and give the URL. Never
  fabricate a release that isn't in the captured text.
- Include real dates/numbers exactly; never round or invent.
