## 官方 / 政策

- [SP-API Release Notes](https://developer-docs.amazon.com/sp-api/docs/sp-api-release-notes)
  - 发生了什么: 2026-04-29 一批 API 变更：External Fulfillment Shipping v2024-09-11 新增 US 国内 Easy Ship 支持、新增 drop-off 地点详情（地址/地图/距离）与美制单位；Fulfillment Outbound v2020-07-01 更新了 getPackageTrackingDetails。
  - 卖家影响: 用 SP-API 做自发货/多渠道履约、轨迹追踪的卖家，External Fulfillment 与 Fulfillment Outbound 对接需按新字段适配；Easy Ship 美国本土支持是新能力。
  - 建议动作: 让对接方核对 ERP/面单系统是否要跟进 External Fulfillment Shipping 与 getPackageTrackingDetails 的字段变更，评估 Easy Ship US 是否可用。

  - 发生了什么: Amazon Ads 官方指南，系统讲 automatic / manual / negative targeting 三类定向方式及如何选择策略。
  - 卖家影响: 这是官方对 SP 定向逻辑的口径，可用来校准你现有自动/手动结构和否定词体系，尤其 Prime Day 前重排广告时。
  - 建议动作: 取一个 ACOS 偏高的 SP campaign 对照指南，检查自动/手动分工是否清晰、否定词是否在拦浪费流量。
